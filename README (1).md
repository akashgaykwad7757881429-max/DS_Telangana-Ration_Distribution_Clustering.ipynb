# Telangana PDS Analytics Dashboard

## Project Overview
This project analyzes Telangana Public Distribution System (PDS) data using data cleaning, exploratory data analysis (EDA), clustering, and dashboard visualization.

## Features
- Data Cleaning
- Missing Value Handling
- District-wise Analysis
- Top Shops Analysis
- KMeans Clustering
- DBSCAN Clustering
- PCA Visualization
- Streamlit Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Streamlit

## Files
- DS_Telangana_Ration_Distribution_Clustering.ipynb
- telangana_app.py
- final_df.csv

## Run the Dashboard

```bash
streamlit run telangana_app.py
